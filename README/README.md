# 🌑⚔️ Shadow Realm VPS Checker ⚔️🌑
**SSH & VPS Credential Validator**
⚡ Fast • Optimized • Stable • Premium

---

## 🚀 Overview
Shadow Realm VPS Checker is a multi-threaded tool that validates SSH/VPS credentials from bulk lists. It tests each server connection and saves all working VPS credentials to a hits file, with live progress and concurrent threading for fast scanning.

---

## 🎮 Features

### 🖥️ Check Features
- Host:User:Password validation
- Host:Port:User:Password support
- Working VPS credential saving
- Large credential list support

### ⚙️ System Features
- ⚡ ThreadPoolExecutor multi-threaded
- 📊 Live valid/invalid counters
- 📁 Hits saved to output file
- 🔄 Timeout and error handling
- 🌑 Clean terminal progress display

---

## 📦 Installation

```bash
1. Download the ZIP file
2. Extract the archive
3. Install dependencies:
   pip install -r "VPS CHECKER requirements.txt"
4. Run the script:
   python "VPS CHECKER.py"
```
